# Datasets

> **Datasets will be available after final publication.**

The event-based datasets used to train and evaluate the system will be released here once the manuscript is published, together with the metadata needed to reproduce the results.

Planned contents of this directory upon release:

- **MNIST digits** — randomly moving targets recorded in transmission through a solid silicone–SiO₂ phantom (OT ≈ 3.3). 10,000 training / 2,000 testing sequences.
- **Kanji (KMNIST) characters** — spatially fixed, time-varying-contrast targets recorded in reflection through the solid phantom (OT ≈ 6.6, ~144 photon mean free paths).
- **Birds** — 50 distinct bird silhouettes, each executing 100 unique naturalistic trajectories (5,000 sequences; 4,200 training / 800 testing), recorded in transmission through a turbid-water tank (diluted intralipid, OT ≈ 3.5).
- **Dynamic fog** — moving targets recorded in a fog chamber with fluctuating optical thickness.

Each release will include raw/preprocessed DVS event streams, ground-truth target images and positions, optical-thickness calibration, and train/test splits.
